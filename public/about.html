The website showcases the Rohingya Refugees settlement data in an interactive map. This website allows us visually inspect the camp location on the map with additional information like the name of the settlement, population and which Upazila and Union the camp belong to. All this data is publicly available and collected from The Humanitarian Data Exchange website.

The app contains three types of data with individual API endpoint

  • Camp location: RAW data
  • Local footpath: RAW data
  • District boundary: RAW data
